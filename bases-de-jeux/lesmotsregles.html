Mode d'emploi

Explication de l'activité et exemples de règles possibles

Quelques pistes pour comprendre le jeu et l'adapter à votre groupe.

Le concept de ce jeu est de partir d'une ou plusieurs lettres pour créer différents mots qui doivent respecter des contraintes. Marquez 1 point par contrainte validée.

  1. Choisissez le nombre de lettres que vous souhaitez.
  2. Cliquez sur « Changer les lettres » pour modifier aléatoirement les lettres (si vous souhaitez garder une lettre, cliquez dessus pour la verrouiller).
  3. Choisissez le nombre de contraintes que vous souhaitez (si vous souhaitez garder une contrainte, cliquez dessus pour la verrouiller).
  4. Cliquez sur « Nouvelles contraintes » pour générer aléatoirement de nouvelles contraintes.
  5. Quand la mise en place vous convient, les participant·e·s commencent et doivent trouver 1 mot par contrainte qui commence par la ou les lettres choisies.
variantes possibles
  • Choisissez plusieurs lettres : elles doivent toutes être contenues dans chaque mot.
  • À partir d'une lettre, les participant·e·s doivent trouver un mot qui répond à toutes les contraintes en même temps. Ils gagnent un point par contrainte respectée.
À retenir

Et comme je le dis pour chaque activité, ce ne sont que des exemples ; surtout, sentez-vous libres de vous servir de ce qui est proposé pour en faire ce qui vous convient le plus.
